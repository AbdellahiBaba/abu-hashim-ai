API Documentation

RESTful API for Arabic AI text generation, model management, and evaluation.

GET /api/health Health check and model status

Response

{
  "status": "healthy",
  "model_loaded": false,
  "version": "1.0.0"
}
POST /api/generate Generate text from prompt

Request Parameters

ParameterTypeRequiredDescription
prompt string Required The input text prompt
temperature float Optional Sampling temperature (0.0 - 2.0, default: 0.7)
max_tokens integer Optional Maximum tokens to generate (default: 512)
top_p float Optional Nucleus sampling parameter (default: 0.9)
stream boolean Optional Enable streaming response (default: false)

Example Request

POST /api/generate
Content-Type: application/json

{
  "prompt": "اكتب قصيدة عن الربيع",
  "temperature": 0.7,
  "max_tokens": 512
}

Example Response

{
  "response": "في الربيع تتفتح الأزهار...",
  "tokens_used": 128,
  "latency_ms": 450
}

POST /api/evaluate Run model evaluation

Request Parameters

ParameterTypeRequiredDescription
test_set string Optional Path to test dataset (default: built-in test set)
metrics array Optional List of metrics to evaluate (default: all)
POST /api/update-model Trigger model update from learning buffer

Description

Processes validated entries from the learning buffer and triggers an incremental training run. This is an admin-only endpoint.

Response

{
  "status": "update_started",
  "entries_processed": 42,
  "estimated_time": "15 minutes"
}
