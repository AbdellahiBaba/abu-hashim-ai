# QalamAI Typography Guidelines

## Primary Fonts

### Arabic Text
- **Font Family**: Noto Kufi Arabic, Amiri, serif
- **Usage**: Arabic content, headings, body text in Arabic

### Latin Text
- **Font Family**: Inter, system-ui, sans-serif
- **Usage**: English content, UI labels, code elements

### Monospace
- **Font Family**: JetBrains Mono, Fira Code, monospace
- **Usage**: Code snippets, API documentation, terminal output

## Font Sizes

| Element         | Size   | Weight | Line Height |
|-----------------|--------|--------|-------------|
| H1              | 2.5rem | 700    | 1.2         |
| H2              | 2rem   | 600    | 1.3         |
| H3              | 1.5rem | 600    | 1.4         |
| H4              | 1.25rem| 500    | 1.4         |
| Body            | 1rem   | 400    | 1.6         |
| Small           | 0.875rem| 400   | 1.5         |
| Caption         | 0.75rem| 400    | 1.4         |

## Color Usage in Typography

- **Headings**: #0A1A2F (Dark Navy) on light backgrounds, #FFFFFF on dark backgrounds
- **Body Text**: #333333 on light backgrounds, #E0E0E0 on dark backgrounds
- **Accent Text**: #D4AF37 (Gold) for highlights, links, and emphasis
- **Muted Text**: #9E9E9E for secondary information

## Direction Support

- RTL (Right-to-Left) for Arabic content
- LTR (Left-to-Right) for English content
- Use `dir="auto"` for mixed content
